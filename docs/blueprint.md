# **App Name**: Mongo Health Hub

## Core Features:

- User Account Creation: Create and manage user accounts for different roles (doctors, nurses, admin) with specific permissions and secure authentication.
- Patient Registration: Register new patients with unique identification, personal information, and summarized medical history.
- Appointment Scheduling: Schedule appointments for general and specialized medicine services. Integrate AI-powered suggestions for optimal time slots based on doctor availability and patient history as a tool.
- Emergency Triage: Enable quick patient registration and classification according to urgency (e.g., Manchester scale).
- Lab Request Management: Automated management of analysis requests triggered from prescriptions.
- Medication Stock Control: In pharmacy module, enable real-time inventory tracking of medications, with alerts for expiration dates and minimum stock levels.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey a sense of trust, stability, and health.
- Background color: Light pale green (#E0F8E0), a very desaturated version of the primary, to create a calm and clean atmosphere.
- Accent color: A muted olive green (#808000) for secondary actions, 30 degrees away from teal on the color wheel to maintain visual harmony while still creating differentiation.
- Body text and headline font: 'Inter', a grotesque-style sans-serif for a neutral and modern feel.
- Use clear, simple icons that represent different medical services and departments within the hospital. Consistent style for easy recognition.
- A clean and structured layout with clear visual hierarchy to ensure ease of navigation and readability. Prioritize important information.
- Subtle transitions and animations for user interactions, such as loading screens, data updates, and form submissions.
